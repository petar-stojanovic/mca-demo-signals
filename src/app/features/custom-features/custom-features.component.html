Custom Store Features

signalStoreFeature lets you extract reusable store behavior — one line to add it to any store.

These are features you write. Not built-in to SignalStore, but built the same way. Drop them into any store with one line.

1 withLogger

Logs every state change to the console. One line to add to any store.

Open DevTools console and interact with the previous pages to see it in action.

2 withRequestStateAndErrorHandling

Tracks loading, success, and error for any HTTP call. Returns a handleRequest operator — pipe any request through it. Uses rxMethod under the hood.
